2 JavaScript条件判断以及循环

2.1 if语句

if(x) {code;code;….} else {code;code;…}

注意：Javascript把null, undefined, 0, NaN 和空字符‘ ’视为false，其它一概是true 所以 if(s.length)为true

if(x){…}else if(){} else{};

注意： 在多个if…else…语句中，如果一个条件成立了就不会继续向下执行了。所以需要 避免定义条件有重合部分.

2.2 循环语句

  • for (i=1; i<=10000; i++){code} (与C语言比较类似）
  • 数组内索引：对于长度可以使用 i<数组.length
  • 数组/ 对象内属性名称索引：for （var 数组/变量 in 对象）
    • 对于数组，得到结果是’0’，’1’，’2’，’3’…格式是string,不是数字
    • 对于对象，得到的是对象名称， 而且也可以得到所有继承的属性。「」对象一章
  • 如果需要过滤掉对象的继承属性，只保留此对象自己的属性，可以使用hasOwnProperty()

2.3 使用while语句

  • 如同C语言，有do与while的先后之分
  • 两者区别是do是先循环，在判断条件，而while是先判断条件，再循环。
  • 使用do while要提前考虑一下。d第一次do以后产生的结果。
