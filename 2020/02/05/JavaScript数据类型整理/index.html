JavaScript数据类型整理(1)

1 JavaScript的数据类型

1.1 一般数据

Javascript不区分整数和浮点数（数据类型一致），表示2进制，8进制以及16进制的方法与C语言一致

  • NaN与所有其它数不等（包括它自己) NaN===NaN //false, 唯一能够判断NaN的方法是通过isNaN(…)
  • 浮点数计算会产生误差 1 / 3 不等于 (1 - 2 / 3)，2数相等，需要加入精度
  • null在JavaScript中表示空，它和0(数值)以及’’(长度为0的字符串)不同
  • undefined用于判断函数参数parameter是否传递有用。

1.2 字符串

和python相同使用" 或者'，在字符串中需要使用转意字符来表示比如 'I\'m \"OK\"!';结果是 I'm "OK"!，和C语言一样,\可以转义其它字符，比如\n表示换行,\t是制表符。ASCII字符可以以\x##的十六进制表示，’\x41’; // 完全等同于 ‘A’, 而\u####表示一个Unicode字符，比如 '\u4e2d\u6587'; // 完全等同于 '中文'。

    1.3 逻辑运算

    与C语言相通，使用&&，｜｜以及！表示与或非

    1.4 比较运算符

    注意：== 会自动转换类型以后再比较，===不会自动转换数据类型。这是JavaScript设计缺陷，所以建议不要使用==进行比较。

    1.5 数组

    和python的list类似
    一个数组可以包含任意数据类型，比如[1, 2, 3.14, ‘Hello’, null, true]; 创建数组也可以使用圆括号()建立(比如: new Array(1, 2, 3) 但是不建议)
  • 数组可以重新被索引赋值，但是没有赋的值将产生undefined。这个和python不同。
  • 复制一个数组：使用slice(),但是两个数组不等。
  • 数组头尾元素添加与删除：
    • push()可以在数组末尾添加元素
    • pop()可以吧array最后的元素删除
    • unshift和shift分别对应在数组头部增加以及删除元素
  • 数组头尾元素添加与删除:
    • Array+Array 连接: concat,返回值是一个新的Array, 如果接受的是多维数组，会降维其中元素整合进入
  • 数组元素顺序处理：
    • 按照一定顺序排序:sort()；
    • 反转所有元素： 从1，2，3变为3，2，1:reverse()
  • 与数组的索引相关的操作
    • 索引位置：使用indexOf和string类似，可以搜寻一个值的位置，返回索引数
    • 索引截取数组部分：slice 与字符串的substtring类似，用来截取部分元素，返回一个新的array
    • 从指定索引开始(包括该索引号)删除若干元素，然后从该位置开始添加元素：splice()
  • 字符串成一体: join()可,和python用法类似

1.6 数组的结构解析: 对数组中每一个元素同时处理

  • 在ES6开始，JavaScript增加了结构解析，可以通过多个变量形式进行赋值
  • 可以出现嵌套以及忽略元素，注意保持嵌套时候的层次
